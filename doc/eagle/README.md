# ServiceFactory Smart Jacket Touch Circuit
#### PCB for Touch Sensing from two Inputs, sending both signals over the OneWire-Bus
###### Components used:
-  2x AT42QT1011   - Capacitive Touch Chip
-  1x DS2413       - OneWire Converter
- 2x Cf 100nF
- 2x Cs 68nF
- 2x Cx 5.6pF
- 2x R0 0Ω
- 1x Rp 4.7kΩ
- 2x Rs 220Ω
